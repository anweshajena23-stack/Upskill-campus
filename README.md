 **Crop Yield Prediction Using Machine Learning**
 **Project Overview**

This project predicts agricultural crop yield using Machine Learning techniques. The model is trained on historical crop production data, including factors such as crop type, state, season, rainfall, fertilizer usage, pesticide usage, cultivated area, and production. The objective is to help estimate crop yield more accurately and support data-driven agricultural decisions.

**Objective**
. Predict crop yield using machine learning
. Analyze the impact of agricultural and environmental factors on crop production.
. Build an accurate regression model for yield prediction.

 **Dataset**
The dataset contains the following features:
. Crop
. Crop Year
. Season
. State
. Area
. Production
. Annual Rainfall
. Fertilizer
. Pesticide
. Yield (Target Variable)

 **Technologies Used**
. Python
. Google Colab
. Pandas
. NumPy
. Matplotlib
. Scikit-learn
. Joblib

** Machine Learning Algorithm**
Random Forest Regressor

 **Project Workflow**
1. Import required libraries.
2. Load and explore the dataset.
3. Handle missing values.
4. Encode categorical variables.
5. Split the dataset into training and testing sets.
6. Train the Random Forest Regression model.
7. Predict crop yield.
8. Evaluate the model using R² Score and Mean Absolute Error (MAE).
9. Visualize feature importance and prediction results.
10. Save the trained model using Joblib.

** Model Performance**
R² Score: 0.9066
Mean Absolute Error (MAE): 10.43

These results indicate that the model predicts crop yield with high accuracy.
